# AI Chatbot for Career Counselling System

## Overview
This project is an AI-powered chatbot designed to assist students and professionals in making informed career decisions. It uses a **Retrieval-Augmented Generation (RAG)** approach to provide accurate and context-aware career guidance. The system leverages embeddings and large language models for intelligent responses.

---

## Features
- Conversational AI interface for career counselling.
- Retrieval-Augmented Generation (RAG) technique for accurate responses.
- Personalized career recommendations using Google Gemini embeddings.
- Responses generated with Groq LLM model for high-performance AI.
- Easy-to-use and interactive interface with Flask backend.

---

## Technology Stack
- **Backend:** Python, Flask
- **RAG (Retrieval-Augmented Generation):** For combining embeddings with LLM responses
- **Embeddings:** Google Gemini
- **LLM:** Groq LLM
- **Frontend:** HTML, CSS, JavaScript
- **Database:** MySQL for user data
